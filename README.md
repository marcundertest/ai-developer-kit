# Integrity Suite

Strict project template for automated setup and code quality.

## Installation

```bash
pnpm install
```

## Usage

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Testing

```bash
pnpm test
```

### Validation

To run all quality checks:

```bash
pnpm test:full
```

## Contribution

1. Fork the project.
2. Create your feature branch.
3. Commit your changes (using conventional commits).
4. Push to the branch.
5. Create a new Pull Request.

## Security

We take security seriously. Run the following to check for vulnerabilities:

```bash
pnpm audit
```
